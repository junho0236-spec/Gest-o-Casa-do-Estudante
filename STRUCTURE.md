# Project Structure

## Overview
This document provides an overview of the project structure and setup instructions for the Gest-o-Casa-do-Estudante repository.

## Directory Structure
- **/src**: Contains the main source code of the application.
  - **/components**: Reusable components used throughout the app.
  - **/pages**: Page components representing different routes.
  - **/styles**: CSS/SCSS files for styling the application.

- **/tests**: Contains unit and integration tests.

- **/public**: Static files such as images and favicon.

- **/docs**: Documentation files for the project.

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone https://github.com/junho0236-spec/Gest-o-Casa-do-Estudante.git
   cd Gest-o-Casa-do-Estudante
   ```

2. **Install Dependencies**
   If using npm:
   ```bash
   npm install
   ```
   Or if using yarn:
   ```bash
   yarn install
   ```

3. **Run the Application**
   To start the development server:
   ```bash
   npm start
   ```
   Or for yarn:
   ```bash
   yarn start
   ```

4. **Access the Application**
   Open your web browser and navigate to `http://localhost:3000` to view the application.
