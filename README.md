# Gest-o Casa do Estudante

## Descrição do Projeto
O Gest-o Casa do Estudante é uma aplicação destinada a gerenciar as operações e a logística de uma casa de estudantes, facilitando o controle de acomodações, reservas e outras atividades essenciais.

## Funcionalidades
- Cadastro e gerenciamento de estudantes
- Reserva de acomodações
- Controle de entradas e saídas
- Relatórios de ocupação
- Gestão financeira

## Instruções de Configuração
1. Clone o repositório:
   ```bash
   git clone https://github.com/ju no0236-spec/Gest-o-Casa-do-Estudante.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd Gest-o-Casa-do-Estudante
   ```
3. Instale as dependências:
   ```bash
   npm install
   ```
4. Inicie o servidor:
   ```bash
   npm start
   ```
